## Quick Start

### Project setup
```
git clone https://github.com/ntutcsie/ntutcsie.git
cd ntutcsie/
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
